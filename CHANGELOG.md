# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- PyPI release preparation
- CI/CD workflow with GitHub Actions
- Pre-commit hooks configuration
- API reference documentation
- Contributing guidelines

## [0.9.0] - 2026-03-22

### Added
- Complete user documentation
  - Getting started guide
  - Installation guide
  - Configuration guide
  - FAQ
  - Tutorials (Hello World, Custom Entity, Custom Item)
- Example projects
  - Hello World example
  - Custom Entity example
  - Custom Item example
  - Custom UI example
- Performance optimization modules
  - LRU Cache for knowledge base
  - Log batch processor
  - Code generation optimizer
- OpenClaw Skills
  - modsdk-game-executor
  - modsdk-log-analyzer
  - modsdk-autofix

## [0.8.0] - 2026-03-22

### Added
- Game executor for in-game code execution
- Real-time log analyzer with error pattern matching
- Auto-fix module for common errors
- Enhanced CLI with autofix command

## [0.7.0] - 2026-03-22

### Added
- Intelligent code completion
- Code smell detection
- Refactoring suggestions
- Best practices checker
- OpenClaw Skills for code completion and refactoring

## [0.6.0] - 2026-03-22

### Added
- Code execution module with sandbox mode
- Debugger with breakpoints support
- Hot reload functionality
- Performance analyzer

## [0.5.0] - 2026-03-22

### Added
- ChromaDB vector store integration
- Semantic search with sentence-transformers
- Hybrid search (keyword + semantic)
- LlamaIndex framework integration
- Incremental knowledge base updates

## [0.4.0] - 2026-03-22

### Added
- Custom template loading and hot reload
- API binding generator (type stubs)
- Event handler generator
- Code quality tools (complexity analysis, lint)

## [0.3.0] - 2026-03-22

### Added
- Skill base class and registry
- API search skill
- Event search skill
- OpenClaw Skill documentation

## [0.2.0] - 2026-03-22

### Added
- Knowledge base data models
- Markdown document parser
- Knowledge base indexer
- Knowledge base retriever with filters

## [0.1.0] - 2026-03-22

### Added
- Project initialization
- Documentation framework
- Game launcher module
- Log capture TCP server
- Log parser
