### 遇到的问题

1. **YAML 解析问题**
   - 问题：GitHub Actions YAML 中的 `on` 字段被解析为布尔值 `True`
   - 解决：使用 `content.get(True, content.get("on", {}))` 兼容两种情况

2. **TOML 解析问题**
   - 问题：pyproject.toml 是 TOML 格式，不能用 yaml.safe_load 解析
   - 解决：使用 Python 3.11+ 的 `tomllib` 模块解析
