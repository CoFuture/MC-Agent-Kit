## 迭代 #71 计划

### 版本
v1.58.0

### 目标
知识库增强与检索优化

### 背景与动机

迭代 #70 已完成插件系统增强。为了进一步提升知识检索能力，需要：

1. **知识库增强**: 改进文档解析和索引
2. **检索优化**: 提升搜索准确性和速度
3. **示例代码**: 增强示例检索和展示
4. **用户体验**: 改进搜索结果的展示

### 功能规划

#### 1. 知识库增强

**文档解析优化**:
- 改进 Markdown 解析器
- 提取代码块和示例
- 识别 API 和事件定义
- 支持多版本文档

**索引结构优化**:
- 统一索引条目（UnifiedEntry）
- 增强元数据（模块、作用域、版本）
- 关联 API 和事件
- 支持增量更新

#### 2. 检索优化

**混合检索**:
- 语义检索（Embedding）
- 关键词检索（BM25）
- 融合排序（RRF）
- 可配置权重

**查询扩展**:
- 同义词扩展
- 拼写纠错
- 模糊匹配
- 中英文互译

**结果重排序**:
- 相关性重排序
- 多样性重排序
- 时效性重排序
- 可组合策略

#### 3. 示例代码增强

**示例检索**:
- 按 API/事件检索示例
- 按功能场景检索示例
- 示例代码高亮
- 可运行示例

**示例库**:
- 内置示例集合
- 示例分类索引
- 示例质量评分
- 用户贡献示例

#### 4. CLI 用户体验

**搜索命令增强**:
```bash
mc-kb search <query> [options]
  --type {api|event|example}  # 类型过滤
  --module <module>            # 模块过滤
  --scope {client|server}      # 作用域过滤
  --format {text|json|markdown}# 输出格式
  --top-k <n>                  # 结果数量

mc-kb api <name>               # 精确查 API
mc-kb event <name>             # 精确查事件
mc-kb example <name>           # 查找示例
```

**结果展示优化**:
- 结构化输出
- 代码高亮
- 关联文档链接
- 使用建议
